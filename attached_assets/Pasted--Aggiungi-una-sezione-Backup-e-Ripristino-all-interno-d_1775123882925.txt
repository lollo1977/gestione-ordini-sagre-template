Il file deve essere scaricato automaticamente con un nome dinamico, es: configurazione_luna_wolfie_[NomeSagra].json.

Funzione Importa: >    - Aggiungi un'area di caricamento file (input type file) con l'etichetta 'Ripristina da Backup'.

Al caricamento del file JSON, l'app deve leggere i dati, sovrascrivere il localStorage e mostrare un messaggio di successo: 'Configurazione ripristinata con successo!'.

Dopo l'importazione, esegui un refresh della pagina (window.location.reload()) per applicare immediatamente le nuove impostazioni.

Sicurezza:

Aggiungi un messaggio di avviso: 'Attenzione: l'importazione di un file di backup sovrascriverà tutti i dati attuali'.

Stile: >    - Usa icone semplici (es. una freccia in su per l'export e una in giù per l'import). Mantieni il design pulito e minimale in linea con il brand Luna Wolfie."
